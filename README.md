# 📈 Trade Explorer (Stock Market Web App)

## 🔰 Project Overview

Trade Explorer is a modern web-based stock market dashboard inspired by platforms like Groww. It allows users to view real-time stock data, analyze market trends, and search for stocks easily.

The project demonstrates the practical use of **JavaScript, API integration, and responsive UI design**.

---

## 🎯 Objective

The main objective of this project is to:

* Integrate a public API to fetch real-time financial data
* Display stock information dynamically on the UI
* Build an interactive and responsive web application
* Apply JavaScript concepts like asynchronous programming and DOM manipulation

---

## 🌐 API Used

This project uses the **Finnhub Stock API**.

* **API Name:** Finnhub Stock API
* **Base URL:** https://finnhub.io/api/v1

## ✨ Features

* 📊 Market Dashboard (Bullish, Bearish, Popular Stocks)
* 🔍 Real-time Stock Search
* 💰 Live Stock Price Display
* 🔃 Sorting (Price & Change)
* 🎯 Filtering (Gainers / Losers)
* 🌙 Dark Mode Toggle


## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Finnhub Public API
