📈 Trade Explorer (Stock Market Web App)

🔰 Project Overview

Trade Explorer is a web-based stock market application inspired by platforms like Groww.
It allows users to view real-time stock data, explore market trends, and search for specific stocks.

The project demonstrates the use of JavaScript, API integration, and responsive UI design.

🎯 Objective

The main objective of this project is to:

* Integrate a public API to fetch real-time financial data
* Display stock information dynamically
* Build an interactive and responsive user interface
* Apply JavaScript concepts like asynchronous programming and DOM manipulation
